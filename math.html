双曲线问题解题过程

已知双曲线 \(C: \frac{x^2}{a^2} - \frac{y^2}{b^2} = 1\) 的离心率为 \(\sqrt{3}\)，点 \((\sqrt{3}, 0)\) 是双曲线的一个顶点。

(1) 求双曲线的方程

由题意可得：

  • a = √3
  • e = c/a = √3
  • c = a * e = 3
根据 \(c^2 = a^2 + b^2\)，可得 b^2 = 6。 所以，双曲线的方程为： \[C: \frac{x^2}{3} - \frac{y^2}{6} = 1\]

(2) 求 AB 的长

设直线 l 的方程为 \(y = \sqrt{3}x + k\)。 将直线 l 的方程代入双曲线方程，得： \[\frac{x^2}{3} - \frac{(\sqrt{3}x + k)^2}{6} = 1\] 化简得： \[x^2 - 2\sqrt{3}kx - 3k^2 - 6 = 0\] 设 A(x1, y1), B(x2, y2)，则 x1, x2 是上式的两个根。 根据韦达定理，有：

  • x1 + x2 = 2√3k
  • x1 * x2 = -3k^2 - 6
又因为点 F(3, 0) 在直线 l 上，所以 k = -√3。 代入上式，得 x1 + x2 = -6，x1 * x2 = -15。 所以，AB 的长度为： \[AB = \sqrt{(x1 - x2)^2 + (y1 - y2)^2} = \sqrt{(x1 + x2)^2 - 4x1x2} = \sqrt{36 + 60} = \sqrt{96} = 4\sqrt{6}\]

因此，AB 的长度为 \(4\sqrt{6}\)。
